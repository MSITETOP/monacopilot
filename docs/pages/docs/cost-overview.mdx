# Cost Overview

Monacopilot utilizes the Groq API for AI auto-completion. The cost of completions is very affordable. Costs vary depending on the model used; see the table below for an estimate of the costs you will need to pay Groq for completions.

| Model Name      | Average cost per 1000 Code Completions |
|-----------------|----------------------------------------|
| llama3-70b-8192 | $0.287                                 |

<Callout type="info">
  The cost is calculated based on the [Groq pricing](https://wow.groq.com/).
</Callout>
